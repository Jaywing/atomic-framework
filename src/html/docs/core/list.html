List

Usage

To apply this component, add the .List class to an unordered or ordered list. The list will now display without any spacing or list-style.

  • List item 1
  • List item 2
  • List item 3
<ul class="List">
  <li>List item 1</li>
  ...
</ul>

Modifiers

List block (.List--block)

  • List item 1
  • List item 2
  • List item 3
<ul class="List List--block">
  <li>List item 1</li>
  ...
</ul>
List block with links
  • List item 1
  • List item 2
  • List item 3
<ul class="List List--block">
  <li><a href="#">List item 1</a></li>
  ...
</ul>
List block with links, without list
List item 1 List item 2 List item 3 
<div class="List List--block">
  <a href="#">List item 1</a>
  ...
</div>
