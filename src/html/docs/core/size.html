Size

Responsive component for determining widths

A utility object that can be combined with other object to determine widths. By default it accommodates up to 10 grid columns. You set different widths for different break points using the responsive modifiers.

Usage

Classes

Class Description
o-Size--full Fills 100% of the available width
o-Size--1of2 Half width
o-Size--1of3 Third width
o-Size--2of3 2 thirds width
o-Size--1of4 Quarter width
o-Size--3of4 3 quarters width
o-Size--1of5 Fifth width
o-Size--2of5 2 Fifths width
o-Size--3of5 3 Fifths width
o-Size--4of5 4 Fifths width
o-Size--1of6 Sixth width
o-Size--5of6 5 Sixths width
o-Size--1of10 Tenth width
o-Size--9of10 9 Tenths width
