Icon

Place vector icons anywhere using our SVG sprite generator.

Usage

Drop icon svgs in the 'icon' folder in 'src' and they are automatically added to the sprite, it's location is indexed with a reference that matches its filename.

You can then use 'icon' macro (located in the 'src/html/macros/helpers.html' file.

{{ icon('[icon_ref]') }}

{{ icon(i.id) }}
{{ i.id }}
{% endfor %}
