File structure

An overview of the top-level file structure of Atomic

gulpfile.js/       # directory for all gulp tasks and settings
|-- lib/           # library of helper scripts and setting for gulp tasks
|-- tasks/         # each gulp task has its own script in here
|-- config.json    # main config file for gulp tasks, if a task has no config in here, it wont run
src/               # directory for all the projects source files
|-- fonts/         # custom web font files go here - don't forget to use webfont-loader to load them
|-- html/          # nunchucks template files
|-- icons/         # drop icon svg's in here to be converted to icon font and svg sprite
|-- images/        # site images - any new ones added will be put through imagemin
|-- javascripts    # modular es6 scripts go here
|-- static         # favicons, app images etc. Gets transferred to root of public
|-- stylesheets    # all the scss files
.travis.yml        # example config for travis-ci (continuous integration tool)
karma.conf.js      # config for karma js testing tools
