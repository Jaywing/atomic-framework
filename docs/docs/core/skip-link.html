Skip link

Hidden skip link that appears when it gains focus (usually by tabbing to it)

You can see an example of the skip link by reloading this page and pressingTAB.

Usage

<a href="#content" class="c-skip-link">Skip to content</a>

The skip link must be the first focus-able element on the page, so the that it is activated by pressing tab for the first time.
