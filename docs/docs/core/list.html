List

Easily create nicely looking lists, which come in different styles

Usage

To apply this component, add the .c-list class to an unordered or ordered list. The list will now display without any spacing or list-style.

  • List item 1
  • List item 2
  • List item 3
<ul class="c-list">
  <li>c-list item 1</li>
  ...
</ul>

Modifiers

List block (.List--block)

  • List item 1
  • List item 2
  • List item 3
<ul class="c-list c-list--block">
  <li>List item 1</li>
  ...
</ul>
List block with links
  • List item 1
  • List item 2
  • List item 3
<ul class="c-list c-list--block">
  <li><a href="#">List item 1</a></li>
  ...
</ul>
List block with links, without list
List item 1 List item 2 List item 3 
<div class="List c-list--block">
  <a href="#">List item 1</a>
  ...
</div>

List divider (.List--divider)

  • List item 1
  • List item 2
  • List item 3
<ul class="c-list c-list--divider">
  <li>List item 1</li>
  ...
</ul>
